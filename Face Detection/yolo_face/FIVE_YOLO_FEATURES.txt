FEATURE 1: MULTI-FACE DETECTION
================================================================================

What it does:
  Detects multiple faces in a single frame, not just one.

How it works:
  The YOLO neural network processes the entire image at once and outputs
  coordinates for ALL detected faces in parallel.

Example:
  Input:  Image with 3 people
  Output: 3 bounding boxes (one per face)
  
  Frame: [в–Ҳв–Ҳв–Ҳв–Ҳ Person 1 в–Ҳв–Ҳв–Ҳв–Ҳ Person 2 в–Ҳв–Ҳв–Ҳв–Ҳ Person 3 в–Ҳв–Ҳв–Ҳв–Ҳ]
                  вҶ“              вҶ“              вҶ“
         BOX1[...] BOX2[...] BOX3[...]

Your implementation:
  вң“ yolodesk processes entire frame
  вң“ Returns coordinates for all faces found
  вң“ Draws all boxes in the output stream
  вң“ Displays count at frame intervals: "frame=X faces=N"

Command line:
  python yolodesk --model yolov8n-face.pt live --source ...

Real output example:
  frame=0 faces=0    (no faces in frame 1)
  frame=30 faces=2   (2 faces found in frame 31)
  frame=60 faces=1   (1 face in frame 61)
  frame=90 faces=3   (3 faces in frame 91)

FEATURE 3: CONFIDENCE SCORES
================================================================================

What it does:
  Provides a probability (0-1) for each detection indicating certainty.

How it works:
  YOLO neural network outputs a confidence value:
  - 1.0 = 100% certain it's a face
  - 0.9 = 90% certain
  - 0.5 = 50% certain
  - 0.1 = 10% certain (likely noise)

Example:
  Detection 1: 0.95 (95% confident - show it)
  Detection 2: 0.60 (60% confident - show it with --conf 0.6)
  Detection 3: 0.45 (45% confident - hide it with --conf 0.6)

Your threshold (--conf 0.6):
  Confidence >= 0.6 вҶ’ Show detection (label and box)
  Confidence <  0.6 вҶ’ Hide detection (ignore)

Impact:
  --conf 0.3: Show more detections (include uncertain ones)
  --conf 0.5: Balanced (default)
  --conf 0.6: Current setting (medium confidence)
  --conf 0.8: Strict (only very confident detections)

Your implementation:
  вң“ Threshold filtering at 0.6
  вң“ Displays percentage on each box: "face: 95%"
  вң“ Adjustable with --conf parameter
  вң“ Real-time filtering of low-confidence false positives

Example output on video:
